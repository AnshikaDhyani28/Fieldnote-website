# Fieldnote — Education Platform Website

A concept website I built for **Fieldnote**, a practical-skills education platform built around one idea: *learning should feel like fieldwork, not a syllabus.*

🔗 **Live site:** https://anshikadhyani28.github.io/Fieldnote-website/

## About

Fieldnote reimagines online learning as a "logbook" — instead of passive video courses, learners ship real weekly projects, get mentor feedback, and build a public record of everything they've made.

## Sections

- **Homepage** — hero intro with a live "learner log" concept
- **About Us** — platform story, mission, and stats
- **Programs** — six course offerings across web dev, data, design, and writing
- **How It Works** — a simple 3-step process
- **Testimonials** — past learner "logbook" entries
- **Mentors** — practitioner-mentor profiles
- **Contact** — enquiry form and contact details

## What This Took To Build

Honestly, most of the effort here wasn't code — it was deciding what the site should actually feel like before touching any HTML. That meant:

- Picking a concept (education-as-fieldwork) instead of a generic template, so every section had a reason to exist
- Writing real content for each section — course names, testimonials, mentor bios — instead of filler "Lorem ipsum" text
- Choosing a consistent visual identity (color palette, fonts, the "stamp"/"logbook" motif) and applying it everywhere instead of defaulting to a stock look
- Structuring the layout in plain HTML/CSS so it works as a single file with no build tools, frameworks, or dependencies
- Making it responsive so it doesn't break on mobile
- Testing the contact form interaction and fixing spacing/alignment issues by eye across sections

## Built With

- HTML5 & CSS3 (single-file, fully responsive)
- Google Fonts (Fraunces, IBM Plex Mono, IBM Plex Sans)
- Vanilla JavaScript (form interaction)
